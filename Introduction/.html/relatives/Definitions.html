Définitions
Autres

Un site web est un ensemble de pages liées entre elles et accessibles via un navigateur web. Un site est composé de plusieurs éléments fondamentaux qui fonctionnent ensemble pour afficher du contenu et permettre des intéractions à l'utilisateur.
Voici de quoi un site web est généralement composé :

  1. Les éléments principaux d'un site web

    1. Structure : le HTML (Hypertext Markeup Langage)

      Le HTML est le squelette d'un site web. Il définit la structure et le contenu des pages. Sa structure est généralement composée de titres et de paragraphes. Elle peut aussi comporter du contenu vidéo ou audio, ou de simples images. Pour cela on utilise des balises.

    2. Apparence : le CSS (Cascading Style Sheets)

      Le CSS s'occupe de l'apparence et du design des pages web. Il permet de choisir les couleurs, les positions, les polices d'écriture, les formes et les tailles. Le CSS est important car il permet de rendre un site attrayant et agréable à utiliser. Il va se combiner au HTML afin de styliser chaque élément.

    3. Dynamisme : le JavaScript

      Le JavaScript rend le site intéractif. Il permet d'ajouter des animations, des calculs, des mises à jour en temps réel etc... Cet élément technique permet de rendre le site plus vivant. Après tout, ce n'est pas un livre !

  2. Les composants d'un site web

    1. Page web

      Une page web est un fichier HTML unique. Vulgairement, elle est pour un site web ce qu'est une page pour un livre. Il peut par exemple y avoir des pages d'accueil ou des pages de contact.

    2. Liens

      Les liens (balise ‹a›) connectent les différentes pages, ou nous dirigent carrément vers d'autres sites. Le "web" (toile en anglais)... Cela fait sens.

    3. Images et fichiers multimédia

      Les images, vidéos ou sons changent du texte et rendent le site plus vivant. Ils sont ajoutés grâce aux balises ‹img›, ‹video› et ‹???›.

    4. Formulaires

      Les formulaires sont des moyens d'interaction pour l'utilisateur. Ils peuvent permettre de s'inscrire, d'envoyer un message, de cocher une case etc...

  3. Le fonctionnement d'un site web

    1. Hébergement

      Un site web doit être hébergé sur un serveur pour être accessible via internet.

    2. Domaine

      L'adresse web permet aux utilisateurs d'accéder au site.

    3. Le navigateur

      Lorsqu'on inscrit une adresse web dans la barre de recherche d'un navigateur, celui-ci demande le contenu au serveur, télécharge le HTML, le CSS et le JavaScript puis affiche la page affiliée à l'adresse web.

Définitions explicatives

  1. Balise

    Les balises sont l'un des éléments de base du langage HTML. Elles agissent comme des étiquettes qui structurent différents éléments d'une page web (texte, image, liens etc...) Souvent il y a une balise ouvrante et une balise fermante. L'ouvrante est composée de son nom, encadré par des chevrons "‹›". La fermante se différencie par un slash "/" juste après le premier chevron.

  2. Attribut

    Les attributs sont des informations supplémentaire incluses dans des balises pour fournir des détails ou configurer le comportement de celles-ci. Ils permettent de personnaliser les éléments HTML, en définissant par exemple leur apparence, leur fonction ou leur intéraction avec les utilisateurs. Un attribut est toujours écrit dans la balise d'ouverture d'un élément HTML, juste après le nom de la balise.

Eléments

Structure

● ‹html›
○ Conteneur principal d'un document html.

● ‹head›
○ Contient les métadonées et les ressources du document.

● ‹title›
○ Définit le titre de la page (onglet du navigateur).

● ‹body›
○ Contient le contenu visible de la page.

● ‹main›
○ Représente la partie principale de la page.

● ‹section›
○ Sert à regrouper les contenus liés.

● ‹header›
○ Représente le contenu d'introduction de la page

● ‹footer›
○ Représente le contenu de "conclusion" de la page

● ‹aside›
○ Représente un contenu non directement lié au contenu principal.

Texte et mise en forme

● ‹h1› à ‹h6›
○ Titres, de niveaux 1 à 6.

● ‹p›
○ Paragraphe.

● ‹br›
○ Saut de ligne.

● ‹hr›
○ Ligne horizontale.

● ‹strong›
○ Texte important (gras).

● ‹em›
○ Texte en italique (emphase).

● ‹u›
○ Texte souligné.

● ‹span›
○ Conteneur en ligne sans signification particulière.

● ‹div›
○ Division ou conteneur pour regrouper des éléments.

Listes

● ‹menu› = Liste de description.
Les éléments de cette liste sont créés avec la balise ‹li›.
● ‹ul› = Liste non ordonnée.
Les éléments de cette liste sont aussi créés avec la balise ‹li›.
● ‹ol› = Liste ordonnée.
Les éléments de cette liste sont aussi créés avec la balise ‹li›.

○ ‹li› = Elémént de liste.

● ‹dl› = Liste de définitions.
○ ‹dt› Texte défini de la liste.
○ ‹dd› = Description ou définition.

Liens et navigation

● ‹a›
○ Lien hypertexte.

● ‹nav›
○ Section de navigation.

Multimédia

● ‹img›
○ Image

● ‹video›
○ Vidéo

● ‹audio›
○ Audio

● ‹source›
○ Source pour ‹video› ou ‹audio›

● ‹track›
○ Sous-titre pour ‹video› ou ‹audio›

Formulaires

● ‹form›
○ Formulaire.

● ‹input›
○ Champ de saisie.

● ‹textarea›
○ Zone de texte multiligne.

● ‹button›
○ Bouton cliquable.

● ‹label›
○ Etiquette pour un champ.

● ‹select›
○ Liste déroulante.

● ‹option›
○ Option dans une liste déroulante.

● ‹fieldset›
○ Groupe de champs.

● ‹legend›
○ Légende pour un ‹fieldset›.

Tableaux

● ‹table›
○ Tableau.

● ‹tr›
○ Ligne de tableau.

● ‹th›
○ En-tête de tableau.

● ‹td›
○ Cellule de tableau.

● ‹caption›
○ Légende du tableau.

● ‹thead›
○ Section d'en-tête.

● ‹tbody›
○ Section du corps.

● ‹tfoot›
○ Section du pied.

Sémantique et structure

● ‹header›
○ En-tête de section ou de page.

● ‹footer›
○ Pied de pâge

● ‹section›
○ Section générique

● ‹article›
○ Article indépendant

● ‹aside›
○ Contenu pour un média

● ‹figure›
○ Conteneur pour un média

● ‹figcaption›
○ Légende pour un ‹figure›

Sripts et resources

● ‹script›
○ Code JavaScript.

● ‹link›
○ Lien vers une ressource externe (CSS, favicon, etc...).

● ‹style›
○ Style CSS intégré.

● ‹meta›
○ Metadonnées

Attributs

- Liste d'attributs globaux -
Attribut Description
id Identifie un élément de manière unique sur la page.
class Associe un élément de manière unique sur la page.
style Associe un élément à un ou plusieurs groupes de styles ou comportements.
title Ajoute une info-bulle lorsqu'on survole l'élément.
lang Définit la langue de l'élément.
hidden Cache un élément sans le supprimer du document.
contenteditable Rend l'élément éditable pr l'utilisateur.
draggable Permet de rendre l'élément déplaçable ("true" ou "false")
spellcheck Active ou désactive la vérification orthographique ("true" ou "false")
accesskey Définit une touche pour accéder rapidement un élément.

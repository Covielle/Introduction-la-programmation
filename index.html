A SAVOIR

La balise ‹title› doit obligatoirement être placée dans la section ‹head› du document HTML. En effet cette balise est conçue pour définir le titre du document qui affiche dans l'onglet du navigateur, les résultats des moteurs de recherche et les favoris ou marque pages enregistrés. Selon la spécification HTML, ‹title› est un élément enfant direct requis dans ‹head›.

La balise ‹style› contient les règles CSS, assurant le design des pages web d'un site. Elle peut être placée directement dans un fichier HTML ou alors, si on souhaite éviter d'encombrer ce fichier, dans un autre fichier entièrement dédié à la partie CSS du site web (fichier CSS externe).
Dans un fichier HTML, cette balise doit être placée dans la section de la balise ‹head›. En effet cette section est chargée en première. Cela permetta alors au site web de charger ses styles avant le rendu du contenu.
